# Module 2: Linux Software and Package Management

## Overview
This module focuses on the operational lifecycle of security software within a Linux environment. The labs contained here demonstrate the ability to provision, audit, and decommission critical security tools using the APT package manager.

## Core Competencies
* **Package Management:** Installing and reinstalling security applications (`Suricata`, `tcpdump`).
* **System Auditing:** Verifying installed software baselines and dependencies.
* **Privilege Management:** Executing administrative tasks using `sudo`.
* **System Hygiene:** Purging unnecessary binaries and verifying removal to maintain a clean security posture.
